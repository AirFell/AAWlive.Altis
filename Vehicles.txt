Barracks: Used for spawning at the bases, changing uniform/kit, and purchasing ammo/weapon crates.
**Note** Each player starts with a default loadout. Each Barracks has 3 levels (upgrades)- 0 is default(spawning only).

	Default Uniform/gear: Guns will be in the same order as Sa'Matras KotH
		West: "B_Soldier_F" remove items: weapons&mags, NV goggles; add default weapons
		East: "O_Soldier_F" remove items: weapons&mags, NV goggles; add default weapons
		Guer: "I_soldier_F" remove items: weapons&mags, NV goggles; add default weapons
